# docker

#### 介绍

ThinkCMF docker环境

通过docker-compose组成容器集群群

#### 目录说明

mysql/data 存放mysql持久化数据

nginx 存放nginx配置

php 存放php配置 php.ini

php-fpm 存放fpm服务的php镜像构建文件

php-swoole 存放swoole服务的php镜像构建文件

php-workerman 存放workerman服务的php镜像构建文件

redis/data 存放redis持久化数据

shell 存放shell脚本

www 存放项目
